== Description ==
# COVID-19 Plugin
## Coronavirus Plugin for WordPress ##

**IMPORTANT:** the plugin is evolving daily, please don't hesitate to add your idea, request changes and features or to report a bug: [https://quarantine.country/coronavirus/plugins/wordpress/issues](https://quarantine.country/coronavirus/plugins/wordpress/issues)

**Coronavirus Update Widget** with coronavirus tracker. Cases by country and by state. Free COVID-19 live update for WordPress, based on multiple sources via [coronavirus API](https://api.quarantine.country). Source code and .zip download soon, see demo:

-  [Coronavirus update USA](https://coronavirus.cases.today/usa/)
-  [Casos de coronavirus en Espa√±a](https://coronavirus.cases.today/spain/)

Coronavirus Update for WordPress is built on the robust and reliable coronavirus data API: api.quarantine.country. By accessing the API, you are not sharing any information. The service is free and does not collect data about you or your WordPress website. Please read and accept the api.quarantine.country [privacy policy](https://quarantine.country/coronavirus/api/privacy.html) to understand and to use the service.
